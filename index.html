About InkStain

InkStain is a concept-driven tattoo portfolio focused on dark and abstract art. This project explores bold compositions and symbolic storytelling through tattoo design.

Built as both a creative and front-end development project, it reflects my journey into design, structure, and user experience.

Based in Malta — currently building a portfolio of original concepts.
